# Company Search (Go + Fiber)

A simple web app that lets users search companies by job roles.

## Run locally

1. Clone the repo
git clone https://github.com/yourusername/company-search.git

2. Go to the folder
cd company-search

3. Install dependencies
go mod tidy

4. Run the server
go run main.go

5. Open in browser
http://localhost:3000
