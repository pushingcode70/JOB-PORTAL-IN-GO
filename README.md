# JOB-PORTAL-IN-GO
# Company Search (Go + Fiber)

A simple web app that lets users search companies by job roles.

## Run locally

1. Clone the repo
git clone https://github.com/pushingcode70/JOB-PORTAL-IN-GO.git
//diretory is JOB-PORTAL-IN-GO/go project
2. Go to the folder
cd "go project"

3. Install dependencies
go mod tidy

4. Run the server
go run main.go
